# MediTrack Agile Planning Document

## 1. Introduction

This document outlines the Agile planning for the MediTrack project, including user stories, product backlog, sprint planning, and reflections on the process.

## 2. User Stories

### 2.1 Functional User Stories

| Story ID | User Story                                                                 | Acceptance Criteria                                                                   | Priority (High/Medium/Low) | Requirement Mapping (Assignment 4) |
